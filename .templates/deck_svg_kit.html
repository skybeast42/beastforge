deck svg kit

In-theme primitives.

Copy a shape into a .col--media (split) or .slide--full. Fills use the deck's CSS vars, so they match the theme on paste. Emphasize one element with var(--accent); invent new shapes in the same idiom.

flow-arrows
A → B → C. A pipeline, a sequence, a seam. Accent the destination.
stack
Layers, precedence, a tier model. Top layer accented = the keystone.
before-after
A change, a migration, a fix. Old muted, new accented.
big-stat
One number that should land. The figure in accent; label beneath.
node-graph
Relationships, a small network, a hub. Accent the hub.
checklist
Done items, a satisfied set of criteria. Accent the ticks.
bars
Growth, a simple comparison, a trend. Tallest/latest accented.
cycle
A loop, an iteration, a feedback cadence. Arrowhead closes it.
funnel
Triage, narrowing, curation. The output tier accented.
annotation-callout
Overlay on an image to mark a region. Ring + leader + label, all accent.
